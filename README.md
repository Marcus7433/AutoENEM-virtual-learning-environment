AutoEnem: AI-Powered Virtual Learning Environment
AutoEnem is conceived as a comprehensive, next-generation Virtual Learning Environment (VLE) designed to democratize access to high-quality exam preparation through Artificial Intelligence. Unlike traditional platforms that rely solely on static content, AutoEnem is built upon a modular and scalable architecture intended to adapt to the student's needs. While the current iteration focuses on linguistic competence, the roadmap includes the imminent integration of modules for STEM (Science, Technology, Engineering, and Mathematics) and Humanities. Future updates will introduce computer vision for solving complex equations and RAG (Retrieval-Augmented Generation) systems for historical and geographical context, transforming the platform into a holistic adaptive learning tutor.

The inaugural module of this ecosystem addresses one of the most significant bottlenecks in education: the correction of essay writing. The "Writing Module" was developed first because it represents the most subjective and time-consuming task for educators. By automating this process, the system provides students with instant feedback on the five official competencies of the ENEM (Brazil's National High School Exam), allowing for rapid iteration and improvement of their argumentative skills without waiting days for a human correction.

The technical workflow begins with the digitization of the student's work. To handle the reality of pen-and-paper exams, the system employs Google Gemini 1.5 Flash, a state-of-the-art multimodal model. This component acts as an intelligent OCR (Optical Character Recognition) transcriber. Crucially, it is prompted not just to read the text, but to preserve the student's original spelling errors, accentuation mistakes, and erasures. This fidelity is essential, as a standard "autocorrect" would mask the very grammatical flaws that need to be evaluated.

Once the text is digitized, the core assessment engine takes over. The numerical scoring is handled by a specialized BERT model (Bidirectional Encoder Representations from Transformers). This model has undergone a rigorous fine-tuning process, transforming it from a general language model into a text regressor. Instead of classifying text into categories, it utilizes a regression head to predict a precise scalar score based on the semantic and structural patterns learned from thousands of graded essays. Finally, to provide the student with actionable advice, the system calls upon Llama-3 (via the Groq API). This Large Language Model analyzes the essay's logic and cohesion, generating detailed, human-like pedagogical feedback that explains the grade and suggests specific improvements.

The entire infrastructure is orchestrated using a modern tech stack designed for performance and separation of concerns. The backend logic and AI processing are built with Python, utilizing Flask as the API gateway and PyTorch for model inference. The user interface is delivered through a responsive Node.js application, ensuring a smooth experience for submitting images and viewing results. Additionally, a dedicated analytics dashboard built with Streamlit allows for the visualization of performance metrics and data trends, completing the loop of the learning environment.
